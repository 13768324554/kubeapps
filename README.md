# Kubeapps Installer

## Installation

Download latest Kubeapps Installer from the [release page](https://github.com/kubeapps/installer/releases)

## Build

```
$ make binary
```

## Usage

```
# Bring up cluster
$ ./kubeapps up

# Bring down cluster
$ ./kubeapps down
```
